Nick Lockwood


@nicklockwood

Nick is a Software Engineer at Schibsted Product and Technology. He has opensourced a lot of popular iOS libraries like iCarousel, FXForms, Layout to name few. He is also the autor of iOS Core Animation: Advanced Techniques

Nick's Talk on Day 1 - 12 noon - 12:35 p.m.

Scaling Storyboards and AutoLayout

Developing UI at scale can be challenging. Nick released an open source library called Layout to solve this problem.

In this talk, Nick is going to share how Layout makes it easy for multiple developers to collaborate on the same screens and components, and to review and merge changes using standard tools. And discuss how you can implementing iOS user interfaces using runtime-evaluated expressions for layout and XML template files.
